# Product Roadmap - ScholarQuest (11+ Prep)

This roadmap outlines planned features to enhance the 11+ preparation experience, focusing on engagement, analytics, and comprehensive subject coverage.

## Core Learning Features

### 1. Verbal & Non-Verbal Reasoning
*   **Description:** Add dedicated modules for Verbal Reasoning (VR) and Non-Verbal Reasoning (NVR), which are critical components of GL and CEM exams.
*   **Implementation:**
    *   Image-based questions for NVR (sequences, matrices).
    *   Code-breaking, sequences, and logic puzzles for VR.

### 2. Timed Practice Mode (Completed)
*   **Description:** Allow users to set custom timers for practice sessions, not just mock exams, to build speed and accuracy.
*   **Implementation:**
    *   [x] Toggleable timer per question (60s default).
    *   [x] "Blitz" mode (answer as many as possible in 1 minute).

### 3. Adaptive Learning Paths
*   **Description:** AI-driven progression that automatically targets weak areas based on historical performance.
*   **Implementation:**
    *   Analyze error patterns (e.g., weak in Fractions).
    *   Suggest daily playlists focusing on these topics.

## Engagement & Gamification

### 4. Gamification System (Partially Completed)
*   **Description:** Increase student motivation through rewards.
*   **Implementation:**
    *   [x] **Badges:** "Math Whiz", "Streak Master", "Early Bird".
    *   **Avatars:** Unlockable avatars based on points/levels.
    *   **Leaderboards:** Weekly/Monthly leaderboards (optional/anonymous).

## Analytics & Reporting

### 5. Detailed Analytics Dashboard
*   **Description:** Visual insights into progress.
*   **Implementation:**
    *   Line graphs showing score trends over time.
    *   Heatmaps for topic mastery (Green = Mastered, Red = Needs Work).
    *   Compare performance against average/target levels for specific schools.

### 6. Parent Dashboard & Reports
*   **Description:** A separate view or email report for parents to track their child's progress without intervening in the game.
*   **Implementation:**
    *   Weekly email summaries.
    *   "Problem Areas" alerts.

## Technical Enhancements
*   **PWA Support:** Make the app installable on tablets/phones.
*   **Offline Mode:** Cache questions for on-the-go practice.
